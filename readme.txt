== Description ==

Beebee 比比小程序提供一系列支持个人资质学习的原创小程序主题。您可以使用这些主题打造支持流量主广告的微信小程序。

我们计划输出5款主流功能的小程序主题。上线进度请关注公众号【APP比比】
- 读书者(书籍租借+书籍下载+书籍交易+书籍阅读+语录+活动+资讯+流量主广告+用户交互/基于Wordpress)
- 淘宝客(服饰搭配专题+商品管理+小程序跳转+淘宝客跳转+内容资讯+流量主广告+用户交互/基于Wordpress)
- 恋爱生(美图管理+表情包管理+聊天文字管理+恋爱教程管理+流量主广告+用户交互/基于Wordpress)
- 大学饭(代跑腿管理+客服管理+交友图文管理+活动管理+内容资讯+流量主广告+用户交互/基于Wordpress)
- 观察家(纯内容资讯管理+流量主广告+用户交互/基于Wordpress)

另外，我们还会推出一款社区主题
- 搞机社(Switch游戏爱好者社区/基于DiscuzQ)

以上，主题均基于开源后台构建，使用此插件可完全对小程序进行样式定义，同时打通公众号、小程序、视频号、小商店。
特别适合大学生作为毕业作品、设计师展示作品以及自媒体用户使用。

在公众号【APP比比】我们提供了详细的操作使用教程，你可以完全傻瓜式的进行学习安装以及自定义设置。
我们也提供收费技术指导。

== Installation ==

1. 上传 `wp-app-beebee`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 进入该插件主页启用 `爱书者`主题
4. 主题启用后如需配套的插件支持会进行提示，按照提示安装相关插件即可
5. 进入该插件主题设置进行相关参数设置，包括菜单、布局、广告等参数设置，公众号【APP比比】提供详细教程
6. 添加专题、书目、语录等主题相关文章内容，公众号【APP比比】提供详细教程
7. 配置小程序端代码，公众号【APP比比】提供详细教程
